# 📊 Finance Analysis Dashboard (Power BI)

## 📌 Overview
This project presents an interactive Power BI dashboard analyzing financial transactions to uncover insights related to revenue, customer segments, and risk patterns.

---

## 🔧 Tools & Technologies
- Power BI
- DAX
- Power Query (ETL)
- Excel

---

## 📂 Dataset
- 50,000+ structured transaction records
- Includes customer segments, transaction types, states, and financial metrics

---

## ⚙️ Key Features
- Built dynamic KPI cards:
  - Total Amount (₹1.67M)
  - Total Transactions
  - Avg Transaction Value
  - Total Fees & Tax
  - Year-over-Year comparison

- Performed data cleaning, transformation, and validation using Power Query

- Created interactive dashboards with:
  - Monthly revenue trends
  - Transaction status analysis (Success, Failed, Pending)
  - Customer segment insights (Retail, Premium, SME, Corporate, Wealth)
  - State-wise performance analysis
  - Gender-based distribution

- Implemented slicers for:
  - Year
  - Occupation
  - Category
  - Dynamic metric selection

---

## 📈 Key Insights
- Retail segment contributes the highest revenue
- Gujarat and Delhi are top-performing states
- Majority of transactions are successful (~85%+)
- Revenue trends show fluctuations across months
